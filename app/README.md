# app

应用目录。后端实现、前端实现和应用配置模板按子目录分离。

| 子目录 | 职责 |
|--------|------|
| `backend/` | 本地后端服务代码；负责 API、状态、持久化、文件元数据和外部算法端口编排 |
| `frontend/` | 电脑端工作台和手机端采集前端 |
| `config/` | 应用配置模板和本地配置命名空间；不放配置加载代码 |

## 边界

- 前端不实现 OCR、LLM、图像处理或字段推断。
- 后端不实现 OCR、图像处理、医学诊断建议或通用病种规则引擎；慢阻肺/呼吸系统入院记录的专病字段抽取、规则分段、字段结果归一化、薄规则质量核验和本地 LLM 调用编排属于后端职责。
- 配置目录不提交真实本机路径、模型路径、患者数据或密钥。
