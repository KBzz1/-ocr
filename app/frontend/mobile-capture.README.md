# mobile-capture

手机端上传页说明。目录名沿用历史命名，MVP 语义是 mobile upload，而不是复杂采集会话。

## 职责

- 扫码进入指定任务的上传页。
- 拍照或选择已有图片。
- 上传一张或多张原图。
- 按上传成功顺序展示页码和图片列表。
- 点击"完成上传"并提示回到电脑端审核。

## MVP 交互

- 页面只需要一个明确的"拍照/选择图片"入口。
- 已上传列表展示页序、上传状态、缩略图或文件名。
- 至少上传 1 张图片后才能完成上传。
- 上传失败时提示用户重新选择或重新拍摄后上传。

## 不负责

- 采集会话状态。
- 会话过期、锁定、取消、修订采集。
- 四边形框选、重新框选、坐标上传。
- 裁剪、透视矫正、去噪、去摩尔纹。
- 拖拽排序、补拍替换某页。
- 字段审核、OCR 文本展示和导出。
