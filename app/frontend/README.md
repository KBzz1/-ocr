# app/frontend

前端应用预留目录。包含电脑端工作台和手机端采集页的未来交付位置。

## 说明文件

- `workstation.README.md`：电脑端工作台。
- `mobile-capture.README.md`：手机端采集页。

## 未来职责

- 展示本地服务状态、访问地址和采集二维码。
- 展示任务列表、处理状态、失败原因和重试入口。
- 展示原图、解析文本、结构化字段和人工审核状态。
- 触发确认和导出。
- 提供手机端扫码采集、拍照选择、四边形框选、页序管理。

## 不负责

- OCR、LLM 字段抽取、图像处理算法。
- 文件系统内部细节。
- 从 schema 或文本自行生成字段候选。
