# app/backend

本地后端服务目录，运行在医生本地电脑上，提供离线 API、状态管理和本地持久化。

## 职责

- 提供本地 API 和后端启动入口。
- 管理采集会话、任务状态、文件元数据和审核结果。
- 通过端口调用外部本地图像、OCR、文档解析和 LLM 字段抽取模块。
- 持久化外部模块结果、人工审核结果、导出文件和本地日志。

## 配置边界

- `settings.py` 是后端配置加载与校验代码。
- `../config/` 是 YAML 配置模板和本地配置命名空间。
- 后端代码不得把真实本机路径、密钥、患者数据路径或模型权重写入仓库。

## 不负责

- 实现 OCR、LLM、图像处理或规则抽取算法。
- 在算法失败时生成替代字段。
- 连接医院 HIS/EMR 或任何云服务。
