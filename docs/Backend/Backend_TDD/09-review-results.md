| ID | 层次 | 行为 | RED 失败点 |
|----|------|------|------------|

| BE-REV-002 | API | 修改字段保存 `final_value`，保留 `original_value` | 原值被覆盖 |

| BE-REV-012 | 集成 | 批量确认和单字段编辑都写入修改历史，保留操作来源 | 批量操作不可追溯 |
