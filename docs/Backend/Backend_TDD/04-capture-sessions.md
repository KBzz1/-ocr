# 后端 TDD — 采集会话收敛

> 当前 MVP 不设计独立采集会话。手机扫码上传直接绑定任务。

| ID | 层次 | 行为 | RED 失败点 |
|----|------|------|------------|
| BE-SES-001 | API | 新建任务直接返回任务上传入口，不创建独立采集会话 | 新建任务仍依赖会话 |
| BE-SES-002 | API | 手机上传接口只校验 task_id 和上传令牌 | 仍要求会话状态 |
| BE-SES-003 | API | 完成上传直接推动任务进入 `processing` | 完成上传仍等待会话 finish 状态 |
