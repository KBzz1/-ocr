# BDD 测试术语表

| 术语 | 说明 |
|------|------|
| 工作站 | 运行在本地 Windows 电脑上的后端服务 + 前端页面 |
| 采集会话 | 手机扫码后创建的上传上下文，绑定一个二维码 |
| 四边形框选 | 手机端拍摄后用户确认的四边形角点坐标，用于标记病历有效区域 |
| 算法模块 | 外部团队交付的图像处理 + OCR + LLM 模块，本系统本地打包调用 |
| Schema | 结构化字段定义文件，含版本号和文书类型标识 |
| 医生 | 电脑端操作用户，负责审核、确认和导出 |
| 采集人 | 手机端操作用户，负责拍照和上传（通常与医生为同一人） |
| 采集二维码 | 电脑端生成的二维码，手机扫码后进入采集页 |
| 结构化字段 | 从病历文书抽取出的字段名-字段值对，如"主诉：头痛3天" |
