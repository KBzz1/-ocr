# 前端 TDD — 导出功能

> PRD: PR-FE-007
> 前端只验证下载触发、完整性检查面板、文件名、Content-Type 和错误提示；Excel/JSON 内容结构由后端测试负责。

| ID | 类型 | 用例 |
|----|------|------|
| FE-EXP-001 | 组件 | 已确认任务显示 Excel 和 JSON 导出按钮 |
| FE-EXP-002 | 组件 | 点击导出按钮先调用完整性检查 API，展示预警面板 |
| FE-EXP-003 | 组件 | 预警面板展示未审核、存疑、为空、未定位来源、空值已确认字段数量 |
| FE-EXP-004 | 组件 | 存在未审核/存疑字段时展示预警文案"存在未审核或存疑字段，导出结果可能不完整" |
| FE-EXP-005 | 组件 | 预警面板提供"继续导出"和"返回审核"按钮 |
| FE-EXP-006 | 组件 | 所有字段已确认/已修改/空值已确认时，完整性检查通过，直接触发下载 |
| FE-EXP-007 | 组件 | 用户选择"继续导出"后调用导出 API 并触发下载 |
| FE-EXP-008 | 组件 | 用户选择"返回审核"后不调用导出 API，保持在审核页 |
| FE-EXP-009 | 组件 | 点击"导出 Excel"调用 `GET /api/tasks/{taskId}/export/excel` 并触发下载 |
| FE-EXP-010 | 组件 | 点击"导出 JSON"调用 `GET /api/tasks/{taskId}/export/json` 并触发下载 |
| FE-EXP-011 | 组件 | 导出失败时显示"导出失败：{原因}"，审核数据不受影响 |
| FE-EXP-012 | 组件 | 导出成功后任务显示最近导出时间和格式 |
