# 前端 TDD — 离线、安全与隐私

> 此模块为跨功能质量门禁，所有 PR 必须通过这些用例。

| ID | 类型 | 用例 |
|----|------|------|
| FE-SEC-001 | E2E | 页面加载时无 CDN、远程字体、远程图片或外部 API 请求 |
| FE-SEC-002 | E2E | `context.setOffline(true)` 下，已本地加载页面仍可使用 MSW mock 完成主要流程 |
| FE-SEC-003 | 组件 | 错误 toast 不展示完整病历原文、身份证号等敏感内容 |
| FE-SEC-004 | 组件 | 文件名中含路径穿越字符时，前端只展示安全文件名，不拼接成本地路径 |
| FE-SEC-005 | E2E | 控制台不输出完整 OCR 文本、字段原文或图片 base64 |
| FE-SEC-006 | 组件 | 未 mock API 请求必须让测试失败，防止误接真实网络 |
