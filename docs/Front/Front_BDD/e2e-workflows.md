# 端到端业务流程

> 包含 FE-E2E-001 ~ FE-E2E-004 完整业务流程

```gherkin
Feature: 端到端业务流程
  作为 医生
  我想要 完成从拍摄/选择图片采集到导出的完整病历处理流程
  以便 将纸质病历转化为结构化数据

  Scenario: 完整流程 — 采集三页病历并审核导出
    # 1. 电脑端启动
    Given 工作站已启动且运行正常
    When 我打开工作台
    Then 显示 "系统已启动" 且任务列表为空

    # 2. 创建采集会话
    When 我点击 "新建采集"
    Then 弹窗生成二维码并显示会话状态、剩余时间（默认 30 分钟）和已上传页数
    And 首页不直接展示 IP、端口或完整采集地址
    And 任务列表新增一条 "采集中" 任务

    # 3. 手机端采集
    When 我用手机扫描二维码进入采集页
    Then 显示采集页面，可以点击 "拍摄/选择图片"

    # 4. 上传三页
    When 我依次拍摄/选择 3 张病历图片
    And 每页调整四边形框选范围后确认上传
    Then 页面列表显示 3 页已上传

    # 5. 完成采集
    When 我点击 "完成采集"
    Then 会话锁定，提示到电脑端查看
    And 任务状态变为 "上传完成"

    # 6. 任务处理
    When 我回到电脑端刷新任务列表
    Then 任务状态变为 "处理中" 后最终变为 "待审核"

    # 7. 审核
    When 我进入审核页
    Then 可以看到原图、解析文本和结构化字段
    When 我修改 "主诉" 字段并失焦
    Then 字段状态变为 "已修改"
    When 我批量勾选 "姓名"、"性别"、"年龄" 字段并点击 "批量确认"
    Then 这三个字段状态变为 "已确认"
    When 我标记 "初步诊断" 为存疑
    Then 字段状态变为 "存疑"
    When "婚育史" 字段为空，我点击 "确认空值"
    Then 字段状态变为 "空值已确认"

    # 8. 确认审核
    When 我点击 "确认审核"
    Then 弹出预警提示显示未审核和存疑字段数量
    When 我选择 "继续确认"
    Then 任务状态变为 "已确认"

    # 9. 导出
    When 我点击 "导出 JSON"
    Then 展示完整性检查预警面板
    When 我点击 "继续导出"
    Then 触发 JSON 文件下载
    When 我点击 "导出 Excel"
    Then 触发 Excel 文件下载
    And 任务状态变为 "已导出"

  Scenario: 算法模块未配置时任务失败且无降级路径
    Given 后端算法模块未配置
    When 采集完成后任务进入处理阶段
    Then 任务状态应该显示为 "失败"
    And 错误原因为 "算法模块未配置，无法生成结构化字段"

    When 我点击该任务
    Then 审核页入口不可用或进入后只显示失败态
    And 显示 "重新处理" 按钮

    When 我查看页面
    Then 不应该出现 "人工补录后继续确认/导出" 的降级路径

  Scenario: 离线环境完整流程验证
    Given 电脑处于完全断网状态
    And 工作站以离线模式启动
    When 我访问工作台
    Then 页面正常加载，无外部资源加载失败

    When 我用手机连接电脑热点
    And 访问采集页
    Then 采集页正常加载

    When 我完成采集并提交
    Then 任务正常创建和处理

    When 我进行审核和导出
    Then 所有操作正常完成

    And 整个过程中不应有任何外部域名请求

  Scenario: 采集后修订并再次完成
    Given 我已完成一次采集且任务状态为 "上传完成"
    When 我在电脑端任务列表点击 "修订采集" 并确认
    Then 任务状态回到 "采集中"
    And 手机端采集页恢复可编辑状态

    When 我在手机端补拍第 2 页并调整页序
    And 我重新点击 "完成采集"
    Then 任务重新进入 "上传完成" → "处理中" → "待审核"
    And 会话 ID 和任务 ID 保持不变

  Scenario: 取消进行中的采集
    Given 存在一个 "采集中" 状态的任务且已上传 2 页
    When 我在电脑端点击 "取消采集" 并确认
    Then 会话状态变为 "已取消"
    And 任务状态变为 "失败"
    And 手机端采集页显示 "采集会话已取消"
```
