# 字段来源核验

> 对应 PRD: PR-FE-005

```gherkin
Feature: 字段来源核验
  作为 医生
  我想要 查看每个字段的来源文本
  以便 快速判断自动抽取结果的可信度

  Scenario: 有来源证据的字段展示来源页码和文本片段
    Given 审核页中 "主诉" 字段有 evidence 和 page_no
    When 我查看该字段
    Then 字段旁应该显示来源页码（如 "第1页"）
    And 应该显示来源文本片段

  Scenario: 点击字段时解析文本区滚动并高亮对应片段
    Given 审核页中 "主诉" 字段有来源证据
    When 我点击该字段
    Then 解析文本区应该自动滚动到对应片段位置
    And 对应文本片段应该高亮显示

  Scenario: 无来源字段显示提示信息
    Given 审核页中 "婚育史" 字段没有 evidence
    When 我查看该字段
    Then 字段旁应该显示 "未定位来源" 标识
    And 应该显示 "需人工确认" 的提示

  Scenario: 点击无来源字段时文本区显示说明
    Given 审核页中 "婚育史" 字段没有 evidence
    When 我点击该字段
    Then 文本区应该显示 "此字段无对应来源文本"
    And 不应该伪造或高亮不存在的文本片段

  Scenario: 来源页码不存在于页面列表时不崩溃
    Given 字段的来源页码指向不存在的第 5 页
    When 我点击该字段
    Then 应该显示 "来源页不可用"
    And 页面不应抛出异常或崩溃

  Scenario: 导出前提示未定位来源的字段数
    Given 审核中有 2 个字段标记为 "未定位来源"
    When 我点击 "导出"
    Then 导出前完整性检查应该提示 "2 个字段未定位来源"
```
