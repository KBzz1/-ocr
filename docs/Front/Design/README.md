# 前端设计素材

本目录保存用户提供的前端视觉参考素材，供实现电脑端工作台、手机采集页、审核页和导出页时对齐布局、品牌元素和交互氛围。

## 素材清单

- `HTML/电脑端首页.html`：电脑端首页 HTML 草稿。
- `图片设计稿/网页端首页ui.png`：电脑端首页视觉稿。
- `图片设计稿/网页端首页+手机扫码弹窗部分ui.png`：电脑端首页和扫码弹窗视觉稿。
- `图片设计稿/手机端主页上半部分（未拍照时）ui.png`：手机端未采集时主页上半部分视觉稿。
- `图片设计稿/手机端主页下半部分完成拍照后页面列表ui.png`：手机端已采集页面列表视觉稿。
- `图片设计稿/手机端框选时ui.png`：手机端调整识别范围视觉稿。
- `重庆大学logo.webp`、`新桥医院logo.jpg`：本地品牌素材。

## 使用边界

- 本目录素材是视觉参考，不替代 `docs/产品PRD.md`、`docs/Front/Front_BDD/`、`docs/Front/Front_TDD/` 或后端 API 契约。
- 若视觉稿与 PRD、BDD、TDD 或后端可执行契约冲突，先以契约为准，并在实现前说明差异。
- 不从图片或 HTML 草稿推断结构化字段、OCR 文本、模型输出或医学结论。
- 未经明确要求，不删除、压缩、改名或覆盖原始素材。
